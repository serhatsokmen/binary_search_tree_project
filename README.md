# binary_search_tree_project

## SORU:
[7,5,1,8,3,6,0,9,4,2] dizisinin Binary-Search-Tree aşamalarını yazınız.
Örnek : root x'tir. root'un sağında y bulunur. Solunda z bulunur vb.
